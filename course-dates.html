Course Dates

Upcoming Course Dates


The calendar below indicates the start dates for our full time and part time classes. If you would like to register for an upcoming class feel free to contact us on 021 556 8464 or email us at info@hccollege.co.za for more information.

Programme Outline:

  • Full time (morning) classes will run between 08h30 to 12h30
  • Full time (afternoon) classes will run between 13h00 to 17h00
  • Part time (weekday evening) classes will run between 17h00 to 20h00
  • Part time (Saturday morning) classes will run between 08h30 to 13h00


    • It is important to please note:

  • Course start dates are dependant on student numbers and may change without notice.
  • Students can enrol for Supported Learning Studies at any time of the year.
